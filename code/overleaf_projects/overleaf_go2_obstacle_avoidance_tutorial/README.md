# Go2 Obstacle Avoidance Tutorial

This Overleaf project explains the current Go2 waypoint obstacle-avoidance algorithm used by the RSSI/IMU project.

Build locally with:

```bash
latexmk -xelatex -interaction=nonstopmode -halt-on-error main.tex
```
