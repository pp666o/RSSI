# Nav2 Go2 Tutorial Overleaf Project

This folder is a complete Overleaf project for the Go2 RSSI/Nav2 tutorial.

Recommended Overleaf settings:

- Main file: `main.tex`
- Compiler: XeLaTeX

The project uses `ctexrep` for Chinese text and only standard TeX Live packages commonly available on Overleaf.
